# 09 — Documentos, Firma e Información (Knowledge)


- **Documentos**: repositorio, edición hojas, dividir PDFs, etiquetas/flujo.
- **Firma electrónica**: campos (firma/nombre/fecha), envío y registro.
- **Información (Knowledge)**: wiki tipo Notion, permisos y publicación.



## Documentos

El Módulo de Documentación (aplicación "Documentos") permite tener una empresa libre de papeles, aglutinando y centralizando todo tipo de documentos para hacer la vida más fácil a los usuarios.


![[parte09_Documentos.png]]



1. **Repositorio y Flujo:**

	- Este módulo sirve como un repositorio centralizado.
	
	- Los documentos subidos a otros módulos, como los que se crean en los proyectos, se vuelcan automáticamente en el Módulo de Documentación.
	
	- Se pueden organizar documentos en carpetas.
	
	
	![[parte09_DocumentosCarpetas.png]]
	
	
	- Se pueden asignar etiquetas al documento, lo cual es útil para configurar si, a partir de ese documento, se debe generar una factura o si debe ser enviado a otro departamento para su validación (definiendo así un flujo).
	
	- Es posible asignar a un documento un contacto o un propietario.
	
	
	![[parte09_PropietarioOAsignar.png]]



2. **Edición de Hojas (Hojas de Cálculo de Odoo):**

	- Odoo tiene integración con sus propias hojas de cálculo.
	
	- Al subir un archivo de Excel, se puede abrir y editar directamente con las Hojas de Cálculo de Odoo.
	
	- Esta característica es excelente para trabajar de manera colaborativa, permitiendo rastrear qué ha modificado cada persona y evitando problemas de compatibilidad o de descargas locales.
	
	
	![[parte09_AbirExcelOdoo.png]]


	
	![[parte09_Excel.png]]



3. **Dividir PDFs:**

	- Odoo incluye una funcionalidad práctica para dividir PDFs.
	
	- Esta herramienta identifica todas las hojas del PDF, permitiendo al usuario partir el documento en las partes que desee, generando múltiples PDFs a partir de uno.
	
	- Firma Electrónica
	
	- La funcionalidad de firma electrónica se vincula con el Módulo de Documentación y agiliza significativamente el proceso de gestión de contratos.
	
	
	![[parte09_OpcionDividir.png]]


	
	![[parte09_VistaDividir.png]]



4. **Campos (firma/nombre/fecha):**

	- Es posible arrastrar y soltar campos dentro de un documento (como un contrato) donde se requiere la firma electrónica.
	
	- Los campos incluyen la firma, el nombre, la fecha, casillas de verificación o texto adicional.
	
	
	![[parte09_CamposFirma.png]]


	
	![[parte09_Firma.png]]


	
	![[parte09_VistaFirma.png]]


	
	![[parte09_FirmarAhora.png]]


	
	![[parte09_FirmarTodo.png]]


	
	![[parte09_Validar_EnviarDocumento.png]]



5. **Envío y Registro:**

	- La aplicación permite solicitar a los clientes que firmen documentos de manera digital.
	
	- Al configurar el contrato, se pueden seleccionar los clientes a quienes se enviará, establecer la validez del documento, enviar recordatorios, y añadir gente en copia.
	
	- El cliente recibe un correo electrónico con un enlace para firmar el documento.
	
	- Una vez firmado, el documento se envía y el usuario puede descargar una copia.
	
	
	![[parte09_Descargar.png]]


	
	![[parte09_DocumentosFirma.png]]
	
	
	
	
	- El módulo permite llevar un control y registro de la documentación y los contratos enviados y firmados.


## Información (Knowledge)

El Módulo de Información (Knowledge) es una de las adiciones más recientes y funciona como una potente base de conocimiento.


![[parte09_Informacion.png]]



1. **Wiki Tipo Notion:**

	- El módulo se inspira y se parece mucho a aplicaciones como Notion.
	
	- Permite acumular todo el conocimiento de la empresa (preguntas frecuentes, documentación, contactos, contratos, guías, etc.) tanto a nivel interno como externo.
	
	
	![[parte09_VistaInformacion.png]]
	
	
	- La herramienta funciona mediante espacios de trabajo (Workspaces).
	
	- Utilizando la barra del slash (/), se puede acceder al editor y crear diversos elementos, incluyendo índices, listas de viñetas, listas numeradas, columnas, o subir archivos y enlaces a vídeos.
	
	- Se pueden crear bases de conocimiento estilo Wikipedia.
	
	- Ofrece un historial de versiones que permite restaurar estados anteriores en caso de cambios no deseados.
	
	- Los artículos creados en Knowledge pueden vincularse directamente con tarjetas o tareas en otros módulos, como Proyectos.
	
	
	![[parte09_NuevoEspacioTrabajo.png]]


	
	![[parte09_OpcionesEspacio.png]]


	
	![[parte09_OpcionesDocumento.png]]



2. **Permisos y Publicación:**

	- Se puede compartir la base de conocimiento con otros usuarios.
	
	- La opción "compartir" permite publicar el contenido en la web, generando un enlace público que se puede compartir o integrar en la página web de la empresa.
	
	- Dentro de las opciones de compartir, también se pueden editar los permisos y la visibilidad de la información
	
	
	![[parte09_Compartir.png]]
