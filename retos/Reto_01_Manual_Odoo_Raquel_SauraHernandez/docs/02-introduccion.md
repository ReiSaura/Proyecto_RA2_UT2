# 02 — Introducción

> **Objetivo:** Entender qué es Odoo y qué se cubrirá en el manual.
#### Odoo como suite modular

Odoo es una potentísima suite de aplicaciones empresariales, descrita como: 

> 'La navaja suiza de la gestión empresarial'.

Es una suite de soluciones empresariales de código abierto. La principal característica de Odoo es que es un sistema altamente modular y es muy fácil de adaptar según nuestras necesidades.

En cuanto a las áreas específicas mencionadas, Odoo ofrece:

|                 Módulo                 |                                                                                                                                                                                                                                               |
| :------------------------------------: | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| CRM (Customer Relationship Management) | Permite convertir contactos en clientes, enviar presupuestos y facturas. Lo que se registra en CRM impacta directamente en ventas y contabilidad.                                                                                             |
|                 Ventas                 | Configura permisos por usuario (por ejemplo, comerciales). Las oportunidades se vinculan con facturación tras el cierre.                                                                                                                      |
|       Contabilidad y Facturación       | Odoo ofrece la capacidad de manejar la contabilidad y la factura electrónica. Permite modificar cómo quieres que luzcan tus facturas.                                                                                                         |
|               Inventario               | Control exhaustivo de la cadena de suministro. Parte del conjunto de soluciones que incluye contabilidad, marketing y más.                                                                                                                    |
|               Marketing                | Incluye marketing por correo, gestión de redes sociales y automatizaciones.                                                                                                                                                                   |
|               Proyectos                | Odoo permite tener un gestor de proyectos. Se presenta en formato de tarjetas y puede configurarse si un proyecto va a ser facturable o no, lo cual es útil porque Odoo puede hacer el cálculo de la facturación controlando las horas.       |
|              E - commerce              | Crea páginas web, tiendas online o plataformas educativas. Incluye chat en vivo y automatización de tareas.                                                                                                                                   |
|               Documentos               | Con el módulo de documentación, vamos a poder tener una empresa libre de papeles y meter ahí todo tipo de documentos. Permite acciones como dividir el PDF y vincularla con la firma electrónica para solicitar firmas digitales de clientes. |
|              Información               | El módulo de información es una alternativa a herramientas como Notion. Permite crear una base con preguntas frecuentes, documentación, contratos, contactos, etc.                                                                            |

#### Ventajas

Sus principales ventajas se centran en su capacidad de integrar soluciones, su economía y su flexibilidad:

- Odoo ofrece una _suite_ de soluciones que cubre absolutamente todo, como CRM, contabilidad, gestión de inventario y marketing.

- Posee la mejor relación calidad/precio y es una de las opciones más económicas del mercado. Ofrece una versión Community completamente gratuita y la versión Enterprise es superconómica.

- Es un sistema modular y personalizable donde todo está conectado en Odoo. Esto permite una integración fluida entre módulos y reduce los costos al tenerlo todo en la misma herramienta.

- Su diseño de código abierto permite la personalización al máximo. Si no se tienen conocimientos de programación, sigue siendo muy intuitivo y permite personalizar módulos *'a golpe de click'*.

- La herramienta está diseñada para la escalabilidad, eliminando la incertidumbre de que se quede corta conforme el negocio o las necesidades crezcan.

- La versión Enterprise ofrece el módulo Odoo Studio para que los usuarios puedan crear sus propias aplicaciones sin saber programar, mediante una interfaz de arrastrar y soltar.

- Cuenta con una interfaz de usuario muy fluida, amigable, y es rápida y fácil de usar.

- A diferencia de otros ERPs de código abierto, Odoo tiene una comunidad muy, muy grande detrás.

#### Consideraciones

Odoo tiene algunas consideraciones importantes que los usuarios deben tener en cuenta, especialmente en relación con el rendimiento y el uso de módulos de la versión Community .

Una consideración es que el rendimiento en grandes volúmenes va a depender mucho del alojamiento (hosting) donde se instale el sitio. Es muy importante tener en cuenta que cuantos más datos se manejen, más recursos se necesitarán para que la aplicación funcione correctamente.

Respecto a la calidad de los módulos, especialmente en la versión Community, los usuarios  pueden acceder a muchos módulos desarrollados por la propia comunidad (similares a los plugins de WordPress). 

Pero la calidad de estos módulos puede ser un poco conflictiva en algunas ocasiones. 

Puede ocurrir que algunos módulos no sean compatibles con una actualización o presenten alguna incompatibilidad que pueda hacer fallar o tirar la aplicación. Por esta razón, si se opta por utilizar la versión Community, se debe estar consciente de que se dependerá mucho de la comunidad para el soporte.
